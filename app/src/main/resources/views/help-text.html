Available Methods

Timing & Sleep

randomSleep(int minMs, int maxMs)

Sleeps for a random duration between minMs and maxMs milliseconds

utils.randomSleep(1000, 3000)

Element Detection

isElementPresent(String selector)

Returns true if element exists in DOM, false otherwise

utils.isElementPresent(".submit-btn")

Wait Operations

waitForElement(String selector, int timeoutSeconds = 10)

Waits for element to be present in DOM

waitForElementVisible(String selector, int timeoutSeconds = 10)

Waits for element to be visible on page

waitForElementClickable(String selector, int timeoutSeconds = 10)

Waits for element to be clickable

utils.waitForElementClickable("#submit", 15)

Element Finding

findElement(String selector)

Returns WebElement matching CSS selector

findElements(String selector)

Returns List<WebElement> matching CSS selector

utils.findElements(".menu-item")

Scrolling

scrollToElement(WebElement element)

Scrolls element into view

scrollToTop()

Scrolls to top of page

scrollToBottom()

Scrolls to bottom of page

JavaScript Execution

highlightElement(WebElement element)

Adds red border to element for debugging

executeJS(String script, Object... args)

Executes JavaScript and returns result

utils.executeJS("return document.title")

Frame & Window Management

switchToFrame(String frameSelector)

Switches context to iframe

switchToDefaultContent()

Returns to main page context

switchToWindow(String windowHandle)

Switches to specific browser window

getWindowHandles()

Returns Set<String> of all window handles

Alert Handling

acceptAlert()

Clicks OK on alert dialog

dismissAlert()

Clicks Cancel on alert dialog

getAlertText()

Returns String text from alert dialog

Page Information

takeScreenshot()

Returns String path to screenshot file

getPageSource()

Returns String HTML source of current page

Usage Notes

• All CSS selectors use standard CSS syntax (#id, .class, element, etc.)
• Default timeout for wait methods is 10 seconds
• Methods throw exceptions if elements not found (except isElementPresent)
• Class uses SLF4J logging for debug information
